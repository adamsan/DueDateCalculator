
## DueDate Calculator

#### Requirements to build
 Java 13, Maven
 
 *I used OpenJDK Runtime Environment (build 13.0.2+8) with Apache Maven 3.6.3*
 
#### Run tests
```
mvn clean test
```
command runs the tests.
#### Generate Reports
```
mvn site
```
command generates the javadoc, the reports (surefire, checkstyle, coverage).
